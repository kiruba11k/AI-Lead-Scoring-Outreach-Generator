# AI Lead Scoring & Outreach Generator

This Apify Actor scrapes Google Maps, scores business leads, and generates AI-powered outreach messages.

## Features
- Google Maps scraping
- Lead scoring
- AI-generated WhatsApp & Email messages
- OpenAI integration (optional)
- Apify Proxy support

## Use Cases
- Sales outreach
- SEO agencies
- Web design leads
- Digital marketing campaigns

## Setup
1. Paste Google Maps search URL
2. Choose max results
3. Add OpenAI API key (optional)
4. Run actor

## Output
Each lead includes:
- Lead score & priority
- WhatsApp message
- Email subject & body
